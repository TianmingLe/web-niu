# Ambient Dream 标签/筛选/双模式（Tab 切换）设计稿

## 目标

在不破坏“沉浸阅读”气质的前提下，把便签从“散落文本”升级为“可分类、可筛选、可快速定位”的内容系统，并提供阅读/编辑双模式（桌面端 Tab 一键切换，移动端长按空白切换）。

## 范围（第一阶段）

### 必做

- 阅读/编辑双模式：Tab 切换（仅在非输入态时生效），移动端长按空白切换。
- 标签系统：多标签 + 颜色；维护全局标签库（Palette）。
- 整理视图：按标签筛选；默认开启“按标签分组”（组内仍分页）。
- 搜索：同时匹配便签正文与标签名，并能定位到分组/分页位置。
- 状态持久化：localStorage（兼容旧数据 `tag: string`）。

### 不做（留到后续）

- 更复杂的统计、推荐、AI 自动打标、导出分享卡片等。

## 核心原则

- 默认处于阅读模式：工具 UI 尽量收起，保持沉浸。
- 编辑能力只在编辑模式出现：避免 UI 噪声。
- 所有“强操作”都有可逆与可提示：与现有撤销/Toast 体系兼容。

## 交互设计

### 1）模式系统（Reading / Editing）

**默认：Reading（阅读模式）**
- 允许：滚动/下一句、音乐控制、打开目录、查看便签。
- 禁止/隐藏：整理工具、筛选工具、删除/撤销等“强编辑工具”入口（或降级为只在编辑模式显示）。

**Editing（编辑模式）**
- 显示：整理分页/分组、筛选、搜索、撤销、删除等工具。
- 便签展示：显示标签胶囊/色点更明显；可快速编辑标签。

**切换规则**
- 桌面端：按下 `Tab` 切换模式。
- 当焦点在 `input/textarea` 时：`Tab` 走原生焦点切换，不切换模式。
- 移动端：长按空白处（例如 600ms）切换模式。
- 视觉反馈：切换瞬间出现 2s 的轻提示（例如“编辑模式”/“阅读模式”），自动淡出。

### 2）标签系统

**便签结构（新）**
- `tags: Array<{ name: string; color: string }>`

**标签库（全局）**
- `tagPalette: Array<{ name: string; color: string; updatedAt: number }>`
- 规则：
  - 标签名大小写不敏感，trim 后去重
  - 若新建标签未指定颜色，从预设色盘循环/随机分配
  - 使用某标签即更新 `updatedAt`，用于最近使用排序

**标签编辑入口**
- 仅在编辑模式：
  - 新增便签时：可选“添加标签”（多选/输入创建）
  - 已有便签：点击标签区域进入编辑（增删改色）

### 3）整理视图：筛选 + 默认分组

**筛选**
- 多选标签（OR 逻辑）：命中任意标签即显示
- 增加“未分类”过滤：`tags.length === 0`
- 搜索与筛选可叠加：先筛选再搜索（或反之，但必须一致并写清）

**默认分组（开启）**
- 将“层（Layer）”从纯分页升级为“分组 + 分页”：
  - Group = 标签名（未分类为一个固定组）
  - Group 内再按 `pageSize` 分页
- 层索引含义：`groupIndex + groupPageIndex`
- 展示：
  - 导航条显示“标签色点 + 标签名 + 该组总数”
  - 点击切换组；组内分页沿用“上一层/下一层”或“上一页/下一页”

### 4）搜索定位（正文 + 标签）

**匹配范围**
- 便签正文（`text`）
- 标签名（`tags[].name`）

**定位**
- 输入回车（或点击“下一个”）：
  - 计算全局匹配列表（保持顺序稳定）
  - 将视图切到目标匹配项所在的 group/page，并聚焦该便签

## 数据与存储

### Storage Keys（新增/调整）

- `ambient-dream-tags-palette`：tagPalette
- `ambient-dream-mode`：reading/editing
- 复用已有 notes 存储（其中 note 结构升级）

### 兼容与迁移

旧数据可能存在：
- `note.tag: string`（单标签）

迁移策略（加载 notes 时）
- 若存在 `tag` 且 `tags` 缺失：
  - 转换为 `tags=[{ name: tag, color: paletteLookupOrDefault }]`
  - 同时写入/更新 `tagPalette`
- 迁移完成后保存新结构（后续不再写 `tag`）

## 边界与异常

- 标签名为空：不创建
- 标签名过长：截断显示（例如 12-16 字符），全名在 title/tooltip
- Palette 过大：上限 60（按 updatedAt 保留最近）
- localStorage 写入失败：走现有 Toast 提示

## 验收标准

- Tab 切换模式：在非输入态稳定切换；在输入框里不影响正常 Tab 行为。
- 默认分组开启：整理视图进入时按标签分组，未分类单独成组。
- 筛选：多选标签可正确过滤；筛选 + 搜索叠加行为一致。
- 搜索：命中正文或标签都可定位到正确组/页并聚焦。
- 刷新：便签标签、标签颜色、Palette 与模式状态不丢。
