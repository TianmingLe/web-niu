# Ambient Dream v2（沉浸式梦境便签桌）

一张“可触摸的梦境便签桌”：中心语录像呼吸一样推进，句子会落成便签；你可以收藏、整理、分组、搜索，并在四季氛围里慢慢回看。

## 功能概览

- 阅读与投递：语录在中心显示，翻页会把当前句子落成便签
- 便签：拖拽、置顶、删除、清空、撤销（部分操作）
- 收藏：把喜欢的句子收进收藏夹，并可“再遇见一条”
- 整理模式：分组 / 搜索 / 分层翻页，便签按当前页重新排布
- 四季氛围（v2）
  - 夏天：全屏萤火虫 + 底部草地/野花近景
  - 秋天：落叶飘落，并会在底部堆积
  - 冬天：雪与冷雾
  - 春天：柔雾与花粉感粒子

## 快速开始

这是一个纯前端项目，使用 Vite 构建与预览（用于 Vercel 一键部署与更稳定的线上加载）。

### 方式 A：本地开发（推荐）

在项目目录执行（首次需要安装依赖）：

```bash
npm install
npm run dev
```

打开（Vite 默认端口）：

- http://127.0.0.1:5173/
  - 若你看到端口不是 5173，以终端输出为准

### 方式 B：构建与本地预览（接近线上）

```bash
npm run build
npm run preview
```

打开：

- http://127.0.0.1:4173/

### 方式 C：部署到 Vercel

- 在 Vercel 导入仓库即可（会识别为 Vite）
- Build Command：`npm run build`
- Output Directory：`dist`

## 操作说明（常用）

- 翻页
  - 鼠标：下滚下一条，上滚上一条
  - 触屏：上滑下一条，下滑上一条
- 模式切换：按 `Tab` 在阅读/编辑模式之间切换
- 整理模式：点击“整理”，进入按分组/搜索/分层浏览
- 退出整理：点击“退出整理”或按 `Esc`

## 音乐

- 页面支持上传本地音频（mp3 / wav / ogg），也可使用仓库内 `public/music/` 的示例文件。
- 音乐播放列表与状态会保存在浏览器本地存储中。
- Vercel 静态站不提供目录列表，因此内置曲库通过 `public/music/manifest.json`（构建时生成）加载。

## 数据存储与导入导出

- 便签、收藏、目录等内容主要存于浏览器的 `localStorage`
- 目录面板里提供导出/导入（文本 / JSON / 备份）能力

## 文件说明

- `index.html`：主入口（样式 + 根节点）
- `src/main.jsx`：主逻辑（React）
- `src/ambient-dream-utils.js`：少量可复用的纯函数（排布/草花布局等）
- `public/comments.json`：默认语录数据
- `public/music/`：示例音乐与曲库清单
