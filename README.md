# Ambient Dream v2（沉浸式梦境便签桌）

一张“可触摸的梦境便签桌”：中心语录像呼吸一样推进，句子会落成便签；你可以收藏、整理、分组、搜索，并在四季氛围里慢慢回看。

## 功能概览

- 阅读与投递：语录在中心显示，翻页会把当前句子落成便签
- 便签：拖拽、置顶、删除、清空、撤销（部分操作）
- 收藏：把喜欢的句子收进收藏夹，并可“再遇见一条”
- 整理模式：分组 / 搜索 / 分层翻页，便签按当前页重新排布
- 四季氛围（v2）
  - 夏天：全屏萤火虫 + 底部草地/野花近景
  - 秋天：落叶飘落，并会在底部堆积
  - 冬天：雪与冷雾
  - 春天：柔雾与花粉感粒子

## 快速开始

这是一个纯前端静态项目，不需要安装依赖。

### 方式 A：用 Python 起本地静态服务（推荐）

在项目目录执行：

```bash
python3 -m http.server 8000
```

打开：

- http://127.0.0.1:8000/ambient-dream-v2.html

### 方式 B：直接双击打开

可以直接打开 `ambient-dream-v2.html`，但部分浏览器对本地 `fetch()` 有限制，建议优先使用方式 A。

## 操作说明（常用）

- 翻页
  - 鼠标：下滚下一条，上滚上一条
  - 触屏：上滑下一条，下滑上一条
- 模式切换：按 `Tab` 在阅读/编辑模式之间切换
- 整理模式：点击“整理”，进入按分组/搜索/分层浏览
- 退出整理：点击“退出整理”或按 `Esc`

## 音乐

- 页面支持上传本地音频（mp3 / wav / ogg），也可使用仓库内 `music/` 的示例文件。
- 音乐播放列表与状态会保存在浏览器本地存储中。

## 数据存储与导入导出

- 便签、收藏、目录等内容主要存于浏览器的 `localStorage`
- 目录面板里提供导出/导入（文本 / JSON / 备份）能力

## 文件说明

- `ambient-dream-v2.html`：主页面（包含主要逻辑与样式）
- `ambient-dream-utils.js`：少量可复用的纯函数（排布/草花布局等）
- `comments.json`：默认语录数据
- `music/`：示例音乐
